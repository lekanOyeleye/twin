# 🤖 AI Digital Twin – Interactive Recruiter Assistant

An AI-powered **Digital Twin of Olalekan Oyeleye** that allows recruiters, collaborators, and visitors to **chat directly with an AI representation of me** to learn about my experience, projects, technical skills, and research interests.

The system combines **large language models, persistent memory, and a conversational interface** to simulate my knowledge, reasoning style, and professional background. Instead of reading a static CV, recruiters can **interactively ask questions and receive contextual responses**.

The application is **fully deployed on AWS** and built using a modern AI system architecture with a frontend interface, backend AI orchestration layer, persistent memory, and automated CI/CD pipelines.

# 🏗️ System Architecture

The project follows a **modular AI application architecture**.

**Frontend**

* Interactive chat interface
* Handles user interactions and displays responses

**Backend**

* API layer that orchestrates the AI system
* Processes user prompts and generates responses

**Memory Layer**

* Stores conversation context
* Maintains dialogue continuity

**Infrastructure**

* Terraform-managed cloud infrastructure
* CI/CD automation via GitHub Actions
* Cloud storage using AWS services
